# Sistema-de-Contabilidade
Sistema de contabilidade automatizado desenvolvido em HTML, CSS e JavaScript, com validação de lançamentos, cálculo de impostos em cascata, depreciação linear de ativos e dashboard em SVG, utilizando LocalStorage para persistência dos dados.

Projeto desenvolvido por *Isabela*.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- SVG
- LocalStorage

## Funcionalidades

- Validação do lançamento duplo
- Cálculo em cascata de impostos
- Depreciação linear de ativos
- Dashboard em SVG
